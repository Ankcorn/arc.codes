# Installation

Open up a terminal:

```bash
npm install -g @architect/cli
```

Create a project folder:

```bash
mkdir testapp
cd testapp
arc init
```

Preview locally:

```bash
arc sandbox
```
> <code>Ctrl + c</code> exits the sandbox

Deploy your code:

```bash
arc deploy
```
