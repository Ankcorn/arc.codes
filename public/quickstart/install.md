# Install Architect

This will go fast!

### Open your terminal

```bash
npm install -g @architect/architect
```

### Check Architect version

```bash
arc version
```


### Create a project folder

```bash
mkdir testapp
cd testapp
arc init
```


### Start the dev server

```bash
arc sandbox
```
> `Cmd / Ctrl + c` exits the sandbox


### Deploy your app

```bash
arc deploy
```
