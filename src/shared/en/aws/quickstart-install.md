# Installation

Open up a terminal:

```bash
npm install -g @architect/cli
```

Create a project folder:

```bash
mkdir testapp
cd testapp
arc init
```

Run `arc sandbox` and check out the fresh new app. (Run <code>Ctrl + c</code> to exit the sandbox.)

Deploy the code:

```bash
arc deploy
```
